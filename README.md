# Graphql-mongo-server
Proyecto de juguete para probar GraphQL

## Ejecución

En ambiente de desarrollo:

```
npm start
```

Una vez finalizado el desarrollo:

Generar código: 
```
npm run build
```

Ejecutar:
```
npm serve
```



## Herramientas

* [GraphQL](https://graphql.org)
* Babel
* Apollo GraphQL
